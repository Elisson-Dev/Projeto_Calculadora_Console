# Projeto_Calculadora_Console
Uma calculadora contruída no puro Dart que tem como missão a evolução contínua das próprias funções.

## Features (V. 0.0.1) - 15/11/2025

1 - Operações de Soma, Subtração, Multiplicação e Divisão;

2 - Interatividade de tela via console;

## Missão

- Criar uma calculadora simples em Dart e evoluí-la para funções mais complexas.

## Como executar

1 - Fazer o download do Software Calculadora Console;

2 - Acessar a pasta do projeto "../Projeto_Calculadora_Console/";

3 - Acessar a pasta do arquivo principal "../bin";

4 - Executar o comando ```dart calculadora.dart```;
